## Explicación de como poder ejecutar el programa

## Requisitos
 
- Python 3.x (sin dependencias externas, solo la librería estándar)

## Cómo ejecutar
 
### Opción 1: Jupyter Notebook
 
1. Abrir una terminal en la carpeta del proyecto
2. Ejecutar el siguiente comando:
 
```bash
jupyter notebook
```
 
3. En el navegador, abrí el archivo `notebook.ipynb`
4. Ejecutá todas las celdas con **Run All** (o `Shift + Enter` celda por celda)


### Opción 2: Desde la terminal (sin notebook)
 
Si queres ejecutarlo como script Python puro, copia el contenido del notebook en un archivo `main.py` y ejecuta:
 
```bash
python main.py
```
